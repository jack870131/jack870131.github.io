My Alibaba Summer of Code (ASoC) Experience with Apache RocketMQ

What is Alibaba Summer of Code?

Same as Google Summer of Code (GSoC), Alibaba Summer of Code is also an open source summer internship hosted by Alibaba Cloud. It is such a young program which has only been held twice so far. Every year from around May to August, Alibaba will pick numbers of issues from each of their open source projects and invite students from all over the world to work those problems out. It is quite the same as the common open source activities, which developers from different places discuss and try to solve project issues together online. This kind of internship will pair one or more mentors with a selected student to work out solutions together and guide him/her through the whole process remotely.

During the whole summer, students will experience several different stages. At the beginning of the event, people who intend to join should browse through the official project list first. Within these projects, potential mentors will launch numbers of issues and mark them as specific to ASoC. After deciding which project to join and which issue to solve, students will then be required to provide a proposal to apply for the selected topic. The proposal should include sections about what the problem is and how it can be solved. The implementation plan need to be as detailed as possible. Furthermore, students are asked to provide project deliverables and timeline, this is also important as potential mentors will evaluate if you are able to plan the development progress reasonably.
